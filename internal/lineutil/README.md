# lineutil

LINE 訊息建構工具，基於 LINE Bot SDK v8。

## LINE API 限制

| 項目 | 限制 |
|------|------|
| 每次回覆訊息數 | 5 則 |
| Quick Reply 按鈕 | 13 個 |
| Flex Carousel bubbles | 10 個 |
| Buttons 動作 | 4 個 |
| 文字訊息長度 | 5000 字元 |
| altText 長度 | 400 字元 |
| Postback data | 300 bytes |
