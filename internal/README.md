# internal/

內部套件目錄。

## 目錄結構

- **bot/** - Bot 模組 (id/contact/course)
- **config/** - 設定管理
- **logger/** - 結構化日誌
- **metrics/** - Prometheus 指標
- **scraper/** - 爬蟲系統
- **storage/** - SQLite 資料層
- **sticker/** - 貼圖管理
- **webhook/** - LINE Webhook 處理
- **lineutil/** - LINE 訊息工具

詳細說明請參考各子目錄的 README。
